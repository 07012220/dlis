Appendix A:
Logical Record Types

A.1 Indirectly Formatted Logical Records

Numeric codes 0-127 are reserved for Public IFLRs. Codes 128-255 are reserved for Private IFLRs.

Figure A-1 defines numeric codes for Public Indirectly Formatted Logical Record Types.

Code Type Desciption Data Descriptor Reference Object Type
0 FDATA Frame Data FRAME
1 NOFORMAT Unformatted Data NO-FORMAT
2-126 ... undefined, reserved --
127 EOD End of Data --
Figure A-1. Numeric Codes for Public IFLR Types
